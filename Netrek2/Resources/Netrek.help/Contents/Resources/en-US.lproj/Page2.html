Netrek Websites

In addition to this manual, there are many websites which provide additional information and help for learning Netrek.

https://networkmom.net/netrek   The dedicated website for this client, including an HTML version of this manual and a link to the Swift source code for this client.

http://www.netrek.org   Primary Netrek website, including clients for other platforms, manuals, information for developers, and history. Check out the “clue guides” section!

http://www.netrek.org/clueguides/dantesdogfightguide.php   A very good Netrek dogfighting manual

https://playnetrek.org  Another Netrek website from the Windows Netrek Client author

http://metaserver.us.netrek.org:1080   Web version of the Netrek Metaserver

https://groups.google.com/forum/#!forum/netrek-forever   Google group for Netrek players.

http://mailman.us.netrek.org/mailman/listinfo/netrek-dev   Netrek developer mailing list, including archives.
